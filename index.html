weatherJS

Simple weather app to pull data using the API from openweatherapp. You can add multiple cities from around the world - just use City, State format to preview current conditions of any city.

tipCalculator

Quick math tip calculator that helps determine price of tip based on quality of service received. You can choose to pay as a party of one, or input the number of guests splitting the bill to divvy up the check.

Live Demo
